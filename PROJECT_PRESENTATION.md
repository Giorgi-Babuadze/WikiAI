# WikiAI V3
## პროექტის პრეზენტაცია

### 1. პროექტის იდეა

`WikiAI V3` არის AI-ზე დაფუძნებული ვებაპლიკაცია, რომელიც Wikipedia-ს საჯარო სტატიებიდან ქმნის პერსონას და მომხმარებელს აძლევს საშუალებას ამ პერსონასთან ინტერაქტიულ ჩატში შევიდეს.

პროექტის მთავარი მიზანია ინფორმაციის გაცოცხლება და ჩვეულებრივი ბიოგრაფიული ტექსტის უფრო საინტერესო, ვიზუალური და კომუნიკაციური ფორმით წარმოდგენა.

### 2. რა პრობლემას აგვარებს

ინტერნეტში ბიოგრაფიული ინფორმაციის მიღება ხშირად ხდება მხოლოდ სტატიკური ტექსტის წაკითხვით. ეს ფორმატი:

- ნაკლებად ინტერაქტიულია
- მომხმარებლისთვის სწრაფად მოსაბეზრებელი შეიძლება იყოს
- არ არის მორგებული სხვადასხვა ასაკის აუდიტორიაზე
- რთულია ინფორმაციის “ცოცხლად” აღქმა

`WikiAI V3` ამ პრობლემას აგვარებს იმით, რომ Wikipedia-ს სტატიას აქცევს საუბრის ფორმატად.

### 3. პროექტის მიზანი

პროექტის მიზანია:

- საჯარო ინფორმაციის უფრო თანამედროვე ფორმით მიწოდება
- AI-ის გამოყენებით ინტერაქტიული გამოცდილების შექმნა
- მომხმარებლისთვის განათლებისა და გართობის შერწყმა
- ერთი და იგივე ინფორმაციის სხვადასხვა აუდიტორიაზე მორგება

### 4. როგორ მუშაობს სისტემა

სისტემის მუშაობის პროცესი ასეთია:

1. მომხმარებელი უთითებს Wikipedia-ის ლინკს
2. აპლიკაცია იღებს ამ გვერდის ძირითად ბიოგრაფიულ ინფორმაციას
3. Gemini AI ქმნის შესაბამის პერსონას
4. იქმნება საუბრის ტონი, თემები და ვიზუალური სტილი
5. მომხმარებელი იწყებს ჩატს ამ პერსონასთან

ასევე შესაძლებელია:

- ორი განსხვავებული ადამიანის არჩევა
- ამ ორ პერსონას შორის დიალოგის ავტომატურად დაგენერირება

### 5. მთავარი ფუნქციები

#### Single Persona Mode

- ერთი ადამიანის Wikipedia გვერდიდან პერსონის შექმნა
- ამ პერსონასთან ჩატის დაწყება
- საუბრის ენის არჩევა
- არჩევა, ვინ დაიწყოს საუბარი

#### Two-Person Mode

- ორი ადამიანის არჩევა
- მათ შორის ავტომატური დიალოგის შექმნა
- წინასწარ მომზადებული ჩატის ჩვენება

#### Audience Mode

აპლიკაციას აქვს ორი აუდიტორიის რეჟიმი:

- `Adult` - ბუნებრივი და სტანდარტული ტონი
- `Pupil` - უფრო მარტივი, მეგობრული და საგანმანათლებლო ტონი

#### Visual Theme

- თითოეულ პერსონას აქვს საკუთარი ვიზუალური სტილი
- ფერები და ფონი აირჩევა პერსონის სფეროს მიხედვით
- მაგალითად: მეცნიერისთვის ლაბორატორიული სტილი, სპორტსმენისთვის დინამიკური სპორტული სტილი

### 6. გამოყენებული ტექნოლოგიები

პროექტში გამოყენებულია:

- `React` - frontend ინტერფეისი
- `Vite` - სწრაფი development გარემო
- `Express.js` - backend API
- `Google Gemini API` - ტექსტისა და პერსონის გენერაცია
- `Wikipedia API` - ბიოგრაფიული ინფორმაციის მიღება
- `CSS` - ვიზუალური დიზაინი და responsive განლაგება

### 7. პროექტის არქიტექტურა

პროექტი იყოფა ორ ძირითად ნაწილად:

#### Frontend

Frontend პასუხისმგებელია:

- მომხმარებლის ინტერფეისზე
- input-ების მიღებაზე
- ჩატის ჩვენებაზე
- ისტორიის შენახვაზე
- თემების და დიზაინის დინამიკურ შეცვლაზე

#### Backend

Backend პასუხისმგებელია:

- Wikipedia ლინკის დამუშავებაზე
- ბიოგრაფიული ტექსტის მიღებაზე
- AI prompt-ების გენერაციაზე
- პერსონის, პასუხების და ფონების შექმნაზე
- შეცდომების და fallback მექანიზმების მართვაზე

### 8. პროექტის უპირატესობები

`WikiAI V3` გამოირჩევა შემდეგით:

- ინტერაქტიული გამოცდილება
- ჩვეულებრივი ტექსტის “ცოცხალ” კომუნიკაციად გარდაქმნა
- სხვადასხვა აუდიტორიაზე მორგება
- მრავალპერსონიანი რეჟიმის მხარდაჭერა
- ვიზუალური თემების დინამიკური გენერაცია
- თანამედროვე და მოსახერხებელი ინტერფეისი

### 9. პრაქტიკული გამოყენება

ამ პროექტის გამოყენება შესაძლებელია:

- საგანმანათლებლო პლატფორმებში
- ისტორიული და კულტურული კონტენტის გაცოცხლებისთვის
- მუზეუმების ან ციფრული გამოფენების პროექტებში
- AI demo/product showcase-ისთვის
- სტუდენტური ან აკადემიური პრეზენტაციისთვის

### 10. გამოწვევები

პროექტზე მუშაობისას მთავარი გამოწვევები იყო:

- Wikipedia-დან მიღებული ინფორმაციის სიზუსტე და სტრუქტურირება
- AI პასუხების ბიოგრაფიულ კონტექსტზე მიბმა
- fallback სისტემის შექმნა, როცა მოდელი დროებით მიუწვდომელია
- პერსონაზე მორგებული ვიზუალური სტილის ავტომატიზაცია

### 11. მომავალი განვითარება

სამომავლოდ შესაძლებელია:

- მეტი ენის დამატება
- რეალური turn-by-turn დიალოგის რეჟიმის განვითარება
- ხმით საუბრის ფუნქციის დამატება
- მომხმარებლის საკუთარი თემების ან კითხვების template-ების დამატება
- admin/dashboard ან analytics მოდულის შექმნა

### 12. დასკვნა

`WikiAI V3` არის პროექტი, რომელიც აერთიანებს:

- ხელოვნურ ინტელექტს
- საჯარო ცოდნის წყაროებს
- თანამედროვე ვებტექნოლოგიებს
- ინტერაქტიულ UX დიზაინს

მისი მთავარი ღირებულება არის ის, რომ ჩვეულებრივი ინფორმაციის წაკითხვას ცვლის უფრო ცოცხალი, პერსონალიზებული და საინტერესო გამოცდილებით.
